Brain Control Wheelchair

Summary

The paper presents a prototype model of a Brain-controlled wheelchair for physically disabled persons using BCI technology. The basic control of the wheelchair is done by human brain signals. To move the wheelchair in any specific direction(Left, Right, Backward, Forward), the user will trigger the motor by focusing in particular direction. Neurosky Mindwave headset pre-processes the detected signals and it will forward to machine learning model which check and match the data with sample data and then sent the desired command towards control circuit to drive motors of wheelchair.This project has an average accuracy due to limitation of stable EEG signal through the Neurosky Mindwave headset.
